# Music Archive Project

## 项目概述

一个静态网页音乐档案应用，展示中国艺术家（主要是周杰伦）的音乐作品。使用纯 HTML、CSS 和 JavaScript 构建，无需构建工具、框架或包管理器。

### 入口点
- `Music-Archive-Project.html` - 模块化版本（引用独立的 `style.css`、`app.js`、`data.js`、`lyrics.js`）
- `Music-Archive-Final.html` - 自包含版本（所有 CSS 内联）

### 运行应用
直接在浏览器中打开任一 HTML 文件即可，无需服务器或构建过程。

## 交互状态设计

### 歌曲行交互状态

| 状态 | 序列号 | 背景 | 左侧竖杠 | 标题 |
|------|--------|------|----------|------|
| **普通** | 灰色 `#555` | 透明 | 隐藏 | 浅灰 `#ddd` |
| **悬停** | 灰色 `#555` | 极淡白 `rgba(255,255,255,0.02)` | 隐藏 | 白色 |
| **选中** | **黄色** `var(--accent)` + 加粗 | 黄色渐变 | **显示** + 发光 | **黄色** |

### 设计原则
- **悬停状态保持安静**：不抢眼，只轻微提示
- **选中状态突出**：序列号变黄、左侧竖杠显示、黄色背景渐变
- **视觉层次清晰**：普通 → 悬停 → 选中，递进式视觉反馈

## 功能特性

### 歌词显示
- **LRC 格式支持**：解析标准 LRC 时间戳格式 `[mm:ss.ms]`
- **逐字高亮**：根据播放进度逐字变蓝色，带发光效果
- **自动滚动**：歌词随播放自动滚动，当前行居中
- **淡入淡出**：上下边缘使用 CSS mask 实现渐变遮罩
- **拖拽调整偏移**：可拖拽调整歌词时间偏移量

### 播放器功能
- 完整的播放控制（播放/暂停、上一首、下一首）
- 四种播放模式（顺序、列表循环、单曲循环、随机）
- 音量控制和静音切换
- 进度条拖动
- 收藏功能（带星星升起动画）

### 本地音乐支持
- 上传本地音频文件
- IndexedDB 持久化存储
- 本地音乐视图管理

## 技术栈

- **HTML5**：语义化标签
- **CSS3**：自定义属性、Flexbox、Grid、CSS Mask
- **JavaScript (ES6+)**：模块化、异步处理
- **IndexedDB**：本地音乐存储
- **外部 API**：iTunes Search API（专辑封面）

## 数据结构

### 艺术家对象
```javascript
{
    id: 'unique_id',
    name: 'Artist Name',
    group: 'FirstLetter',  // 字母索引
    category: 'Genre',
    avatar: 'path/to/image',
    albums: [
        {
            title: 'Album Title',
            year: 'YYYY-MM',
            cover: 'path/to/image.jpg',
            songs: ['Song 1', 'Song 2', ...]
        }
    ]
}
```

### LRC 歌词格式
```
[mm:ss.ms]歌词内容
[00:12.36]一步两步三步四步
[00:13.99]望著天 手牵手
```

## 添加新歌曲歌词

### 方法一：添加到 `lyrics.js`
```javascript
'周杰伦-新歌': `[ti:新歌-周杰伦]
[ar:周杰伦]
[00:10.00]歌词第一句
[00:15.00]歌词第二句
`
```

### 方法二：直接添加 LRC 文件
1. 创建 `.lrc` 文件（如 `新歌-周杰伦.lrc`）
2. 将文件添加到项目根目录
3. 在 `lyrics.js` 中引用该文件
