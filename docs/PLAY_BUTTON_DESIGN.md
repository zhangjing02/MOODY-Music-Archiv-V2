## 📊 CSS 代码

### 动画核心

```css
/* 图标切换动画 - 使用弹性缓动 */
transition: all 0.4s cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

**缓动函数说明**:
- `0.68` - 控制动画速度曲线
- `-0.55` - 超调效果（先放大后收缩）
- `0.265` - 返回到正常速度
- `1.55` - 结束时的缓动

**效果**: 图标会有一个巧妙的"弹跳"效果

### 播放时脉冲动画

```css
@keyframes play-pulse {
    0%, 100% {
        filter: drop-shadow(0 0 0 rgba(102, 126, 234, 0));
        transform: scale(1);
    }
    50% {
        filter: drop-shadow(0 0 8px rgba(102, 126, 234, 0.4));
        transform: scale(1.05);
    }
}
```

### 暂停时闪烁动画

```css
@keyframes pause-shimmer {
    0%, 100% {
        filter: brightness(1);
        transform: scale(1);
    }
    50% {
        filter: brightness(1.2);  // 变亮
        transform: scale(1.05);      // 微微放大
    }
}
```
